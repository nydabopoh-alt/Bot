# Diller — игровой Discord-бот

Discord-бот со ставками, мини-казино и ареной боёв. Slash-команды, покер, кости, дуэли, перетягивание каната, интервенция 21.5.

---

## 1. Установка

```bash
# Клонировать репозиторий (если ещё не клонирован)
cd Diller-discord-bot

# Установить зависимости
npm install
```

---

## 2. Переменные окружения

Создайте файл `.env` в корне проекта (скопируйте из `env.example`):

```bash
cp env.example .env
```

Заполните в `.env`:

| Переменная | Описание |
|------------|----------|
| `BOT_TOKEN` | Токен бота из [Discord Developer Portal](https://discord.com/developers/applications) |
| `REDIS_URL` | URL Redis (например `redis://localhost:6379`) |
| `DEFAULT_RTP` | RTP для игр (0.96 = 96%) |
| `GLOBAL_COMMISSION` | Комиссия казино (0.02 = 2%) |
| `APPLICATION_ID` | ID приложения бота (из Developer Portal) |
| `GUILD_ID` | ID сервера (опционально, для быстрого деплоя команд) |

---

## 3. База данных и Redis

```bash
# Сгенерировать Prisma-клиент
npx prisma generate

# Применить миграции (создать таблицы)
npx prisma migrate dev
```

Убедитесь, что **Redis** запущен (локально или удалённо). Бот использует Redis для сессий игр и кэша.

---

## 4. Деплой slash-команд

```bash
npm run deploy
# или
npx tsx src/scripts/deployCommands.ts
```

Команды появятся на сервере (если указан `GUILD_ID`) или глобально.

---

## 5. Запуск бота

```bash
# Режим разработки (с автоперезагрузкой)
npm run dev

# Продакшен
npm run build
npm start
```

---

## 6. Настройка через `/settings`

После запуска бота пригласите его на сервер и выполните `/settings` (доступно только админам и владельцу).

### Каналы

**Основные каналы** — укажите ID или `#упоминание`:
- **Канал логов** — логи ставок, балансов, игр
- **Канал рейтинга** — топ бойцов (белый флаг, командный, похоронный)
- **Канал объявлений** — крупные победы
- **Голосовой канал мероприятий** — для ивентов

**Ставки / Выдача / Игры:**
- **Канал игр** — панель с кнопками игр (кости, покер, дуэль и т.д.)
- **🎟️ Канал ставки** — тикеты, баланс игроков, вывод/пополнение
- **⚖️ Канал выдачи** — админская бухгалтерия

**Админ / Арена:**
- **Канал админа** — меню с кнопкой «Создать событие»
- **Арена** — публикация боёв и турниров
- **Канал событий** — анонсы покера, перетягивания каната

**Авто-обновление рейтинга** — интервал обновления (выключено / минута / час / день).

### Роли

- **Игрок** — роль для доступа к играм (опционально)
- **Модератор** — для модерации
- **VIP** — множитель выигрышей

### Экономика

- **Стартовый баланс** — баланс новых игроков
- **Ежедневный бонус** — награда за `/daily`
- **Мин. / макс. ставка** — лимиты для игр
- **Комиссия с выигрыша** — процент в пользу казино

### Лимиты

- Макс. ставок в минуту
- Макс. выигрыш в час
- Порог крупной победы (для объявлений)

### Игры

Выберите, какие игры доступны на сервере: покер на кубиках, покер (холдем), дуэль, перетягивание каната, интервенция 21.5, покер 2 vs 3 кости, столы.

### Доступ

Режим доступа к игровым командам: только в канале игр или в разрешённых каналах (whitelist).

---

## Полезные команды

| Команда | Описание |
|---------|----------|
| `/settings` | Настройки бота (админы) |
| `/balance` | Проверить баланс |
| `/daily` | Ежедневный бонус |
| `/top` | Рейтинг бойцов |
| `/table` | Создать стол / лобби |

---

## Очистка БД

Скрипт очистки рейтинга, статистики игр и сброса балансов:

```bash
npx tsx src/scripts/cleanRating.ts
```

Каналы и настройки ServerConfig сохраняются.
